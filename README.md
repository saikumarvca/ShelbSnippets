# Shelb Snippets

Turbo productivity toolkit for full-stack + desktop ERP development.

Quick start:

```bash
npm install
npm run compile
# Open in VS Code and press F5 to debug the extension
```

Features:
- Snippets for many frameworks
- Generators for CRUD, ERP modules, and Tauri desktop modules
- Workspace-aware detection
- Tree view snippet explorer
- Analytics (local)

See CHANGELOG.md for updates.
